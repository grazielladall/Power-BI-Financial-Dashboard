# Power-BI-Financial-Dashboard

* Ficticious Company* 

Financial dashboard using Figma and Power BI to communicate main business insights such as:

* Cash Flow Control over time
* Profit margin (%)
* Expenses by department
* Balance (profit vs loss) over time
